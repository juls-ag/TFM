# Autogyro Modeling in Simulink

This project focuses on the development of an autogyro model in Simulink, based on an initial helicopter simulation model.

## Project Structure

- `HelicopterModelingSimulation.slx`  
  Base model (provided helicopter simulation)

- `AutogiroModel.slx`  
  Modified model for autogyro simulation (work in progress)

## Objective

The objective of this project is to adapt a helicopter dynamic model to represent the behavior of an autogyro, considering:

- Autorotation of the rotor
- Modified aerodynamic forces
- Differences in lift generation
- System dynamics adaptation

## Tools

- MATLAB R2025b
- Simulink

## Status

Work in progress — current focus is on adapting rotor dynamics and system behavior from helicopter to autogyro.

## Notes

The helicopter model is used as a reference and baseline. All modifications are implemented in the autogyro model.
